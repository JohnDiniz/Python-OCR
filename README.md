# Python-OCR
extract text from images for linux

### how to use

```sh
python3 main.py
```

### install requirements

```sh
pip install -r requirements.txt
```

for arch

```sh
sudo pacman -S tesseract tesseract-ocr-eng maim
```
### in sxhkdrc 

```
super + shift + p
	python3 ~/Python-OCR/main.py
```
